A simple RISC-V kernel

What features does it have?

Heap allocation and deallocation, multithreading, semaphores, thread sleep and join, console output and input

What is it lacking?

It does not support multoprocesor systems, nor does it have a file system. It does not have a shell, and is not extremely efficient. It is not a full operating system, but rather a kernel that can be used to run a freestanding program linked with it.

Project kernel for the Operating Systems 1 class



Kernel realised in RISC-V made as the final project for the 13S112OS1 class during my second year of studies at the School of Electrical Engineering, University of Belgrade. Works on monoprocessor systems and has full support for memory management, multithreading, semaphores, thread sleep and join functionality. Used to run a freestanding program linked with the kernel.
